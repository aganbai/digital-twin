## 待规划需求

> 以下需求待 V2.0 完成后，根据实际情况决定是否启动 V3.0。

### BL-007: 多账号关联学生身份

| 项目 | 说明 |
|------|------|
| **优先级** | P1 |
| **来源** | V2.0 迭代3 用户反馈 |
| **说明** | 部分学生没有微信，通过家长账号登录。需要允许多个微信账号关联到同一个学生分身。家长拥有全部权限（可查看对话记录、作业等）。核心方案：新增 `persona_bindings` 表，支持 owner（创建者）和 linked（关联者）两种绑定类型。学生生成"关联码"，家长输入关联码绑定到学生分身。 |
