## 需求列表

### BL-001: 知识库支持 URL 导入

| 项目 | 说明 |
|------|------|
| **来源** | 产品需求 |
| **优先级** | P1 |
| **角色** | 教师 |
| **记录日期** | 2026-03-28 |

**需求描述**：
允许教师在添加知识库文档时，除了手动输入文本内容外，还可以输入一个 URL 链接，系统自动抓取并解析该 URL 的网页内容，提取正文后存储为知识库文档。

**初步要点**：
1. 在添加文档页面（FE-P8）新增"URL 导入"入口（Tab 切换或独立按钮）
2. 教师输入 URL 后，后端自动抓取网页内容
3. 后端需实现网页内容解析：去除 HTML 标签、提取正文、处理编码
4. 解析后的内容自动填充到文档标题和内容字段，教师可编辑后提交
5. 需处理常见异常：URL 不可达、内容为空、超时、内容过长等

**涉及改动范围（预估）**：
- **后端**：新增 URL 解析接口（如 `POST /api/documents/import-url`），引入网页抓取和内容提取逻辑
- **前端**：添加文档页增加 URL 输入模式，展示解析结果预览
