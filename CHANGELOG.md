# 数字分身项目变更日志

## V2.0 迭代13 (2026-04-09)

### 新增功能

#### 教材配置流程重构
- **班级创建整合教材配置**: 在班级创建页面新增可折叠的教材配置区域
- **班级编辑支持教材配置**: 编辑班级时可查看、修改、删除教材配置
- **移除独立配置入口**: 个人中心移除"教材配置"菜单，废弃独立配置页面

#### UI优化调整
- **小程序TabBar调整**: 学生端简化为3项(聊天/发现/我的)，教师端调整为4项(聊天/学生/知识库/我的)
- **对话页优化**: 移除导航栏右上角"详情"按钮，语音按钮改为白色喇叭图标
- **+号面板增强**: 新增"新会话"入口，支持学生向同老师新建对话
- **教师端对话页区分**: 显示学生昵称，增加接管/退出接管按钮

### 详细变更

#### 后端变更
- 扩展班级创建API (POST /api/classes)，支持传入 curriculum_config 字段
- 扩展班级更新API (PUT /api/classes/:id)，支持更新教材配置
- 扩展班级详情API (GET /api/classes/:id)，返回关联的教材配置数据
- 保持数据模型不变，教材配置仍通过 persona_id 绑定

#### 小程序端变更
- 班级创建页(class-create/index.tsx)新增教材配置折叠区域
  - 学段选择器(Picker)，支持5种学段类型
  - 年级联动选择(根据学段动态显示)
  - 学科多选标签组件
  - 教材版本多选标签
  - 自定义教材输入(大学及以上学段)
  - 教学进度输入框
- 班级编辑页(class-edit/index.tsx)支持教材配置查看和修改
- 个人中心页(profile/index.tsx)移除"教材配置"菜单项

#### H5教师端变更
- Classes.vue 创建/编辑弹窗新增教材配置折叠面板
- 使用 Element Plus Collapse 组件实现折叠效果
- 适配班级API，支持 curriculum_config 参数传递

#### 数据结构
```json
{
  "curriculum_config": {
    "grade_level": "primary|middle|high|university|adult_life|adult_professional",
    "grade": "年级",
    "subjects": ["学科1", "学科2"],
    "textbook_versions": ["版本1", "版本2"],
    "custom_textbooks": ["自定义教材"],
    "current_progress": "教学进度描述"
  }
}
```

### 技术细节

#### 学段与年级映射
| 学段 | 年级选项 |
|------|----------|
| primary | 一至六年级 |
| middle | 七至九年级 |
| high | 高一至高三 |
| university/adult_* | 无年级选项，支持自定义教材 |

#### 兼容性处理
- 旧版本客户端不传 curriculum_config 时API正常响应
- 已有教材配置数据通过编辑班级可继续访问
- curriculum-config 页面代码保留但标记为废弃

## V2.0 迭代12 (2026-04-08)

### 新增功能

#### 会话管理增强
- **用户中断能力**: 在SSE流式回复时支持点击中断按钮停止AI回复
- **会话列表入口**: 聊天页左上角增加会话列表入口，展示当前分身下所有会话
- **新会话按钮移动**: 将新增会话按钮移到会话列表顶部
- **指令系统**: 支持#开头的指令如#新会话、#给老师留言

#### 技术架构
- **后端接口**: 新增SSE流式中断接口 `/api/chat/stream/:session_id/abort`
- **前端组件**: 新增SessionSidebar会话列表侧边栏组件
- **状态管理**: 增强聊天页面的流式状态管理

### 详细变更

#### 后端变更
- 新增流式中断API端点，支持实时中断SSE连接
- 扩展消息类型支持，识别teacher_message_direct类型消息
- 优化会话管理逻辑，支持更灵活的会话切换

#### 前端变更
- 聊天页新增"停止生成"按钮，替换流式回复时的发送按钮
- 实现SessionSidebar组件，支持会话列表展示和切换
- 导航栏左上角集成会话列表入口图标
- 指令系统支持#新会话、#新对话、#新话题、#给老师留言等指令
- 优化流式回复的用户体验，支持中断后继续对话

#### UI/UX优化
- 侧边栏滑入/滑出动画效果
- 当前会话高亮标识
- 新会话按钮移至侧边栏顶部，提升易用性
- 中断按钮采用红色主题色，突出显示

### 技术细节

#### 中断功能实现
- 使用AbortController API实现SSE流式连接中断
- 中断后保留已接收内容，状态正确重置
- 支持中断后继续正常对话流程

#### 会话管理
- 侧边栏采用遮罩层设计，点击遮罩层可关闭
- 会话列表按时间倒序排列，显示最后一条消息预览
- 支持会话切换时的消息列表实时更新

#### 指令系统
- 前端预检查指令，指令不作为普通消息发送
- 支持指令前后空格，不区分中英文标点
- 指令执行后输入框自动清空
